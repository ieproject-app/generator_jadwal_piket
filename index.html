Cara Deploy ke GitHub Pages:

  1. Buat repository baru di GitHub
  2. Upload file HTML ini ke repository
  3. Ke Settings → Pages → Source: pilih "Deploy from a branch"
  4. Pilih branch "main" atau "master" dan folder "/" (root)
  5. Tunggu beberapa menit, aplikasi akan tersedia di username.github.io/repository-name
